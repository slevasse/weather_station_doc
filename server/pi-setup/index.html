    Setup the Pi

    Pi maintenance

    sudo apt-get update
sudo apt-get upgrade
sudo reboot

    Headless SSH connection

    Source

    1. In the OS SD card
    2. Go to boot cd /Volumes/boot
    3. (Optional) Add a empty file called ssh touch ssh
    4. Add/edit a file called wpa_supplicant.conf 
      country=US # Your 2-digit country code
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
   ssid="YOUR_NETWORK_NAME"
   psk="YOUR_PASSWORD"
   key_mgmt=WPA-PSK
}
    5. Boot the Pi
    6. Connect to the Pi via ssh ssh -Y pi@raspberrypi
    7. The default password is raspberrypi

    Change the default password and create a new user

    • Once connected over ssh, open the software config tool sudo raspi-config
    • Change the password.
    • Also change the boot option to make sure you autolog with the pi user

    Setup the Mosquitto brocker

    The mosquitto broker is light weight and simple to setup. -> Good for my rpi4. source

    • Install sudo apt install -y mosquitto mosquitto-clients
    • enable at boot sudo systemctl enable mosquitto.service

    Mosquitto MQTT server

    • Reboot the server sudo service mosquitto restart

    Setup to have a static IP address

    source * I used .43. because it is on my phone router, probably need to change that when hooking to my home router. 

    sudo nano /etc/dhcpcd.conf

interface wlan0

static ip_address=192.168.43.243
static routers=192.168.43.1
static domain_name_servers=192.168.43.1

    Install the LAMP server

    Source

    1. sudo apt update
    2. Install the server sudo apt install apache2
    3. Install PhP sudo apt install php -y
    4. Restart the apache server sudo service apache2 restart
    5. install mariadb (mysql) sudo apt install mariadb-server php-mysql -y
    6. Restart the apache server sudo service apache2 restart
    7. Secure the sql database sudo mysql_secure_installation
    8. Create a new user for the database 
      pi@raspberrypi:/var/www/html $ sudo mysql --user=root --password
> create user admin@localhost identified by 'your_password';
> grant all privileges on *.* to admin@localhost;
> FLUSH PRIVILEGES;
> exit;
    9. Install phpmyadmin sudo apt install phpmyadmin -y
    10. enable mysqli sudo phpenmod mysqli
    11. Restart the apache server sudo service apache2 restart
    12. Move the files to the server folder sudo ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin
    13. login phpmyadmin @ http://SERVERIP/phpmyadmin , use the username: admin and the password you set before.

    Test the install

    1. Type hostname -I in a terminal of the pi to get its IP address
    2. In a webbrowser type connected to the local network type the IP you got from the previous step e.g.192.168.43.243
